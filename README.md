# 서울 지하철 이용량 분석 프로젝트

서울열린데이터광장에서 제공하는 공공데이터를 이용해  
지하철 역별 이용량을 분석하는 Python 프로젝트입니다.

## 📊 사용 데이터

- subway.csv  
  (서울열린데이터광장 — 지하철 승·하차 인원 데이터)

## 🔍 주요 기능

- CSV 데이터 불러오기
- 역별 총 이용객 수 계산
- 혼잡역 TOP 10 출력
- Bar Chart 시각화

## 🚀 실행 방법

### 1. 설치

pip install pandas matplotlib

shell
코드 복사

### 2. 실행

python analysis.py
